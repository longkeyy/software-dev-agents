# Claude Code Agents 工作流程指南

## 产研VP统筹协调

当需要完整的产品开发流程协调时，请使用 **vp-engineering** 智能体。该智能体负责：
- 端到端的产品开发生命周期管理
- 跨职能团队协调和资源分配
- 技术架构决策和技术栈选择
- 敏捷开发流程和最佳实践实施

## 三阶段工作流程

### 🎯 阶段1：分析与设计
**参与团队**: 产品部门 + 架构团队 + UX设计 + 安全团队
- 需求分析和可行性评估
- 系统架构和技术方案设计  
- 用户体验设计和原型制作
- 安全和合规要求定义

### 🔨 阶段2：开发与测试
**参与团队**: 后端 + 前端 + 算法 + 数据 + QA团队
- 功能开发和代码实现
- 单元测试和集成测试
- 代码审查和质量保证
- 持续集成和自动化测试

### 🚀 阶段3：部署与运维
**参与团队**: DevOps + 基础设施 + 安全 + 运营团队
- 生产环境部署和发布
- 监控告警和性能优化
- 安全加固和合规审计
- 用户反馈和迭代优化

## 智能体协作模式

### 单一任务
针对特定技术问题，直接调用相应专家：
```
特定技术问题 → 领域专家智能体 → 解决方案
```

### 跨职能协作
复杂功能需要多个团队配合：
```
复杂需求 → vp-engineering协调 → 多团队并行 → 集成交付
```

### 全流程管理
完整项目从零到一：
```
项目启动 → vp-engineering统筹 → 三阶段执行 → 持续迭代
```
